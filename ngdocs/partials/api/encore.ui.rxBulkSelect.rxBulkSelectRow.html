rxBulkSelectRow
directive in module encore.ui

Description

A directive you place on your element which will contain the bulk-select checkbox. This directive draws the checkbox itself. This directive takes row as an attribute, pointing to the object representing this row.

Dependencies

  • rxBulkSelect

Usage

as attribute
<ANY rx-bulk-select-row
     row="{Object}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
rowObject

The object representing this row, i.e. the left side of the ng-repeat
