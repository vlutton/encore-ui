rxActiveUrl
directive in module rxActiveUrl

Description

This is used as an attribute to directive <rx-product-resources>, in the form of <li rx-active-url="/servers">. The directive checks if the attribute value is a subset of the current URL. If so it returns the class name "selected".

Usage

as attribute
<ANY rx-active-url>
   ...
</ANY>

Directive info

  • This directive creates new scope.
