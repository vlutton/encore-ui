rxRadio
directive in module rxRadio

Description

Attribute directive that wraps a native radio element in markup required for styling purposes.

Usage

as attribute
<ANY rx-radio
     [ng-disabled="{Boolean}"]>
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
ng-disabled
(optional)
Boolean

Determines if control is disabled.

(default: false)

Example

<input rx-radio />
