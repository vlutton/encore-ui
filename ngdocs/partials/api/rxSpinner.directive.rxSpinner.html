rxSpinner
directive in module rxSpinner

Description

Renders a spinner animation on the provided element given the 'toggle' attribute is truthy

Usage

as attribute
<ANY rx-spinner
     size="{String}"
     toggle="{Boolean}"="{Boolean}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
sizeString
  • Controls the size of the spinner. The options are default (no size specified), mini, small, large and extra-large
toggleBoolean
  • When true, the spinner will display
rxSpinnerBoolean
  • When present, adds an extra class to the spinner depicting the color
