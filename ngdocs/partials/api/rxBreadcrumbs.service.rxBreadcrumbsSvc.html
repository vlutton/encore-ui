rxBreadcrumbsSvc
service in module rxBreadcrumbs

Description

rxBreadcrumbsSvc provides various methods to manipulate breadcrumbs.

Methods

  • setHome(path, name)

    By default, the first breadcrumb will always have an URL of '/' and a name of 'Home'. This can be changed with the rxBreadcrumbsSvc.setHome method.

    It takes the new path as the first argument, and an optional name as the second argument. If you don't pass the second argument, it will reuse whatever name is already there (i.e. 'Home'). The breadcrumb name can contain HTML (ie. '<strong>Home</strong>').

    Parameters
    ParamTypeDetails
    pathString

    This is the relative path within app.

    name
    (optional)
    		String

    This will be the display name.

    Example

    breadcrumbsService.setHome = function (path, name) {
  breadcrumbs[0] = {
    path: path,
    name: name || breadcrumbs[0].name
  };
};
