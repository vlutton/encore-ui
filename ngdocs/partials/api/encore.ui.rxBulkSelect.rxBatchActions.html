rxBatchActions
directive in module encore.ui

Description

This directive is responsible for adding the batch action menu link inside a table header. It can only be used when rxBulkSelect is also present. It should be placed in a element.

It will also transclude <li> items, each representing a modal bulk select action. You don't need to include the correctly styled

    , it will do this for you.

Dependencies

  • rxBulkSelect

Usage

as element:
<rx-batch-actions>
</rx-batch-actions>

Directive info

  • This directive creates new scope.
