rxNestedElement
service in module rxMisc

Description

Helper function to aid in the creation of boilerplate DDO definitions required to validate nested custom elements.

Usage

rxNestedElement([opts]);

Parameters

ParamTypeDetails
opts
(optional)
Object
  • Options to merge with default DDO definitions
PropertyTypeDetails
parentString
  • Parent directive name (i.e. defined NestedElement is an immediate child of this parent element)

Returns

Object

Directive Definition Object for a rxNestedElement

Example

angular.module('myApp', [])
.directive('parentElement', function (rxNestedElement) {
  return rxNestedElement();
})
.directive('childElement', function (rxNestedElement) {
  return rxNestedElement({
     parent: 'parentElement'
  });
});
