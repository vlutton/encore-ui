rxActionMenu
directive in module rxActionMenu

Description

Component to add a clickable cog which brings up a menu of configurable actions.

Normally the menu is dismissable by clicking anywhere on the page, but this can be disabled by passing an optional global-dismiss="false" attribute to the directive.

Usage

as element:
<rx-action-menu
       [global-dismiss="{Boolean}"]>
</rx-action-menu>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
globalDismiss
(optional)
Boolean
  • optional attribute to make menu dismissable by clicking anywhere on the page

(default: true)
