rxFavicon
directive in module rxFavicon

Description

This updates the href of an element, and replaces it with the path to a different image based on the environment.

Usage

as attribute
<ANY rx-favicon="{Object}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
rxFaviconObject

This is a congifuration object for each environment.

PropertyTypeDetails
staging
(optional)
String

This defines the favicon file for the staging environment.

local
(optional)
String

This defines the favicon file for the local environment.

Example

<link rel="icon"
    type="image/png"
    href="favicon.png"
    rx-favicon="{ staging: 'staging-favicon.png', local: 'local-favicon.png' }" />
