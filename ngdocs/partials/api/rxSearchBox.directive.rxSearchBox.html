rxSearchBox
directive in module rxSearchBox

Description

The rxSearchBox directive behaves similar to the HTML "Search" input type. When the search box is not empty, an "X" button within the element will allow you to clear the value. Once clear, the "X" will disappear. A disabled search box cannot be cleared of its value via the "X" button because the button will display.

Usage

as element:
<rx-search-box
       ng-model="{String}"
       [ng-disabled="{Boolean}"]
       [ng-placeholder="{String}"]>
</rx-search-box>

Parameters

ParamTypeDetails
ng-modelString

Model value to bind the search value.

ng-disabled
(optional)
Boolean

Boolean value to enable/disable the search box.

ng-placeholder
(optional)
String

String to override the default placeholder.

(default: 'Search...')

Example

<rx-search-box ng-model="searchModel"></rx-search-box>
