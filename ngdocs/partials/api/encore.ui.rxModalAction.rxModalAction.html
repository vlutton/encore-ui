rxModalAction
directive in module encore.ui

Description

Link which will show a modal window on click, and handle callbacks for pre/post modal actions

Usage

as element:
<rx-modal-action
       [pre-hook]="{function}"
       [post-hook]="{function}"
       [template-url]="{string}"
       [disable-esc]="{string}">
</rx-modal-action>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
[preHook]function

Function to call when a modal is opened

[postHook]function

Function to call when a modal is submitted (not called when cancelled out of)

[templateUrl]string

URL of template to use for modal content

[disable-esc]string

If the disable-esc attribute is present, then "Press Esc to close" will be disabled for the modal. This attribute takes no values.
