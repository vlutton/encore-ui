TokenInterceptor
service in module rxTokenInterceptor

Description

Simple injector which will intercept http request and inject the Rackspace Identity's token into every http request.

Dependencies

  • Session

Example

angular.module('encoreApp', ['encore.ui'])
    .config(function ($httpProvider) {
        $httpProvider.interceptors.push('TokenInterceptor');
    });
