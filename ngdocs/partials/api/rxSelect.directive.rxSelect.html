rxSelect
directive in module rxSelect

Description

This directive is to apply styling to native <select> elements

Usage

as attribute
<ANY rx-select
     [ng-disabled="{Boolean}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
ngDisabled
(optional)
Boolean

Determines if control is disabled.

(default: false)

Example

<select rx-select ng-model="demoItem">
  <option value="1">First</option>
  <option value="2">Second</option>
  <option value="3">Third</option>
</select>
