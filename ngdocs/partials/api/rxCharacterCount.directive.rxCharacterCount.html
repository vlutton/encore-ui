rxCharacterCount
directive in module rxCharacterCount

Description

Monitors the number of characters in a text input and compares it to the desired length.

Usage

as attribute
<ANY rx-character-count
     [low-boundary="{Number}"]
     [max-characters="{Number}"]
     [highlight="{Boolean}"]>
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
low-boundary
(optional)
Number

How far from the maximum to enter a warning state

(default: 10)

max-characters
(optional)
Number

The maximum number of characters allowed

(default: 254)

highlight
(optional)
Boolean

Whether or not characters over the limit are highlighted

(default: false)

Example

<textarea ng-model="model" rx-character-count></textarea>
