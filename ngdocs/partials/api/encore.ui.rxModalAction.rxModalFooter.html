rxModalFooter
directive in module encore.ui

Description

Define a footer for the next modal.

Usage

as element:
<rx-modal-footer
       state="{string}"
       [global]="{string}">
</rx-modal-footer>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
statestring

The content will be shown in the footer when this state is activated.

[global]string

If the global attribute is present, then this footer can be used in other modals. This attribute takes no values.
