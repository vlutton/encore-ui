rxToggleSwitch
directive in module encore.ui

Description

Displays an on/off switch toggle

Usage

as element:
<rx-toggle-switch
       [disabled]="{boolean}"
       [post-hook]="{function}"
       true-value="{expression}"
       false-value="{expression}">
</rx-toggle-switch>

Parameters

ParamTypeDetails
[disabled]boolean

Indicates if the input is disabled

[postHook]function

A function to run when the switch is toggled

trueValueexpression

The value of the scope property when the switch is on

(default: true)

falseValueexpression

The value of the scope property when the switch is off

(default: false)

Example

<rx-toggle-switch ng-model="foo"></rx-toggle-switch>
