Apply
filter in module rxSelectFilter

Description

Used to apply an instance of SelectFilter to an array.

Merely calls the applyTo() method of a SelectFilter instance to an input array. 

<tr ng-repeat="item in list | Apply:filter">

Usage

In HTML Template Binding

{{ Apply_expression | Apply:filter }}

In JavaScript

$filter('Apply')(list, filter)

Parameters

ParamTypeDetails
listArray

The list to be filtered.

filterObject

An instance of SelectFilter
