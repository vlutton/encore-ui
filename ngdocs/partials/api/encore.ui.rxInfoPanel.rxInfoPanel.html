rxInfoPanel
directive in module encore.ui

Description

Renders a generic, pre-styled, info panel, with an optional title.

Usage

as element:
<rx-info-panel
       title="{String}">
</rx-info-panel>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
titleString
  • Text to be displayed in the title area of the info panel
