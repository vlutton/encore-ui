rxSortEmptyTop
filter in module encore.ui

Description

Filter that moves rows with an empty predicate to the top of the column in ascending order, and to the bottom in descending order.

Usage

In HTML Template Binding

{{ rxSortEmptyTop_expression | rxSortEmptyTop }}

In JavaScript

$filter('rxSortEmptyTop')()

Example

[{ name: { firstName: 'Adam' } }, { }] | rxSortEmptyTop 'name.firstName':false
Will sort as [{}, { name: { firstName: 'Adam' } }].

[{ name: { firstName: 'Adam' } }, { name: { firstName: null } ] | rxSortEmptyTop 'name.firstName':true
Will sort as [{ name: { firstName: 'Adam' } }, {}]
