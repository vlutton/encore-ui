rxFormSection
directive in module rxForm

Description

Structural element directive used for layout of sub-elements.

By default, all rxField elements will display inline (horizontally). If you wish to display rxField elements in a stacked manner, you may place the stacked attribute on rx-form-section

Display:
block (full width of parent)
Parent:
rxForm
Siblings:
Any HTML Element
Children:
  • rxField
  • HTML DIV Element

Usage

as element:
<rx-form-section
       [stacked="{*}"]>
</rx-form-section>

Parameters

ParamTypeDetails
stacked
(optional)
*

If present, rxField children will stack vertically rather than display horizontally.

Example

...
<form rx-form>
  <rx-form-section>
    <rx-field>...</rx-field>
    <div>...</div>
  </rx-form-section>
</form>
...
