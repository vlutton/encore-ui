rxPermission
directive in module rxPermission

Description

Simple directive which will show or hide content based on whether or not the user has the specified role. See the rxPermission component demo for an example.

Dependencies

  • Permission

Usage

as element:
<rx-permission
       role="{String}">
</rx-permission>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
roleString
  • Name of required role.
