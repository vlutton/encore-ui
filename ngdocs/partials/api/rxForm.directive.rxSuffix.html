rxSuffix
directive in module rxForm

Description

Structural element directive used to wrap content to be placed inline with a form control element.

  • Best placed after a form control element.
Display:
inline block (only as wide as necessary for content)
Parent:
rxInput
Siblings:
  • rxPrefix
  • Any HTML Element
Children:
Any HTML Element

Usage

as element:
<rx-suffix>
</rx-suffix>

Example

...
<form rx-form>
  <rx-form-section>
    <rx-field>
      <rx-field-name>Salary:</rx-field-name>
      <rx-field-content>
        <rx-input>
          <rx-prefix>$</rx-prefix>
          <input type="number" />
          <rx-suffix>Million</rx-suffix>
        </rx-input>
      </rx-field-content>
    </rx-field>
  </rx-form-section>
</form>
...
