Permission
service in module rxPermission

Description

Simple service for accessing roles and permissions for a user.

Dependencies

  • Session

Methods

  • getRoles()

    Fetch all the roles tied to the user (in the exact format available in their Session token).

    Returns
    Array

    List of all roles associated to the user.

  • hasAllRoles(roles)

    Checks if user has every role in given list.

    Parameters
    ParamTypeDetails
    rolesArray

    List of roles to check against

    Returns
    Boolean

    True if user has every role in given list; otherwise, False.

  • hasRole(roles)

    Check if user has at least one of the given roles.

    Parameters
    ParamTypeDetails
    rolesArray

    List of roles to check against

    Returns
    Boolean

    True if user has at least one of the given roles; otherwise, False.

Example

Permission.getRoles() //returns an array of roles for a user
Permission.hasRole(role) //returns true/false if user has specified role
