rxMultiSelect
directive in module encore.ui

Description

A multi-select dropdown with checkboxes for each option

Usage

as element:
<rx-multi-select
       ng-model="{string}"
       [options]="{Array}">
</rx-multi-select>

Parameters

ParamTypeDetails
ng-modelstring

The scope property that stores the value of the input

[options]Array

A list of the options for the dropdown
