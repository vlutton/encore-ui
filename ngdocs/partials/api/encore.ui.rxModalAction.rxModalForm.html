rxModalForm
directive in module encore.ui

Description

Responsible for creating the HTML necessary for modal form

Usage

as element:
<rx-modal-form
       title="{string}"
       [subtitle]="{string}"
       [is-loading]="{boolean}"
       [submit-text]="{string}"
       [cancel-text]="{string}"
       [return-text]="{string}"
       [default-focus]="{string}">
</rx-modal-form>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
titlestring

Title of modal window

[subtitle]string

Subtitle of modal window

[isLoading]boolean

True to show a spinner by default

[submitText]string

'Submit' button text to use. Defaults to 'Submit'

[cancelText]string

'Cancel' button text to use. Defaults to 'Cancel'

[returnText]string

'Return' button text to use. Defaults to 'Return'

[defaultFocus]string

default focus element. May be 'submit' or 'cancel'. Defaults to 'firstTabbable'
