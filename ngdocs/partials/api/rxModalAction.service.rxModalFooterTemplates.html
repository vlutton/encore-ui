rxModalFooterTemplates
service in module rxModalAction

Description

A cache for storing the modal footer templates This is used internally by rxModalFooter, which is preferred for registering templates over direct calling of this api.

Example

rxModalFooterTemplates.add("step1", "<p>Step 1 Body</p>");
rxModalFooterTemplates.flush(); // returns html string to be inserted into DOM
