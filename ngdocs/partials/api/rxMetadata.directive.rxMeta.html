rxMeta
directive in module rxMetadata

Description

Display:
block (full width of parent)
Parent:
  • rxMetadata
  • SECTION element
Siblings:
Any HTML Element
Children:
Any HTML Element

Usage

as element:
<rx-meta
       [label="{String}"]>
</rx-meta>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
label
(optional)
String

Label for metadata definition/value

Example

<rx-metadata>
  <section>
    <rx-meta label="fubar">
      out of working order; seriously, perhaps irreparably, damaged
    </rx-meta>
  </section>
</rx-metadata>
