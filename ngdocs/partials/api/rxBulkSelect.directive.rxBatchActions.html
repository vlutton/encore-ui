rxBatchActions
directive in module rxBulkSelect

Description

This directive is responsible for adding the batch action menu link inside a table header. It can only be used when rxBulkSelect is also present. It should be placed in a <th> element.

It will also transclude <li> items, each representing a modal bulk select action. You don't need to include the correctly styled <ul>, it will do this for you.

Dependencies

  • rxBulkSelect

Usage

as element:
<rx-batch-actions>
</rx-batch-actions>

Directive info

  • This directive creates new scope.

Example

<th colspan="10">
    <rx-batch-actions>
        <li>
            <rx-modal-action
                template-url="templates/suspend-modal.html"
                controller="SuspendServersCtrl"
                classes="msg-info">
                <i class="fa fa-fw fa-power-off msg-info"></i>
                Suspend Selected Servers
            </rx-modal-action>
        </li>
    </rx-batch-actions>
</th>
