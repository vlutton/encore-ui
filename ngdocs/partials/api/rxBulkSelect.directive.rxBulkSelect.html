rxBulkSelect
directive in module rxBulkSelect

Description

A directive you place on your <table> element to enable bulk select. This directive will automatically add <tr bulk-select-message> into your , which will appear whenever items are selected, and disappear when none are selected. The main responsibility of this directive is to provide a controller for other bulk-select-related directives to interact with.

<table rx-bulk-select
       bulk-source="servers"
       selectedKey="rowIsSelected">
</table>

The directive is also responsible for adding a row to the table header that indicates how many rows are selected and contains buttons to select or deselect all the rows at once.

Usage

as attribute
<ANY rx-bulk-select
     bulk-source="{Object}"
     selected-key="{String}"
     [resource-name="{String}"]>
   ...
</ANY>

Directive info
