rxMetadata
directive in module rxMetadata

Description

Parent directive used for styling and arranging metadata information.

Display:
block (full width of parent)
Parent:
Any HTML Element
Siblings:
Any HTML Element
Children:
  • rxMeta
  • SECTION element

It is highly recommended to use <section> blocks for purposes of arranging information into columns.

Each <section> will be 300px wide and will wrap and stack vertically if the container isn't wide enought to accommodate all sections in a single row.

Usage

as element:
<rx-metadata>
</rx-metadata>

Example

<rx-metadata>
  <section>
    <rx-meta label="fubar">
      out of working order; seriously, perhaps irreparably, damaged
    </rx-meta>
  </section>
  <section>
    <rx-meta label="Foo">Bar</rx-meta>
  </section>
  <section>
    <rx-meta label="Bang">Biz</rx-meta>
  </section>
</rx-metadata>
