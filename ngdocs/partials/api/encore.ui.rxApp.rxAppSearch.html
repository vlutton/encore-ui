rxAppSearch
directive in module encore.ui

Description

Creates a search input form for navigation

Usage

as element:
<rx-app-search
       [placeholder]="{string}"
       [model]="{*}"
       [submit]="{function}">
</rx-app-search>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
[placeholder]string

Title of page

[model]*

Model to tie input form to (via ng-model)

[submit]function

Function to run on submit (model is passed as only argument to function)
