rxStatusHeader
directive in module rxStatusColumn

Description

Place this attribute directive on the <th> for the status columns. It ensures correct styling.

For the <th> component representing the status column, add the rx-status-header attribute, i.e.

<th rx-status-header></th>
Note that status columns are sortable with rxSortableColumn, just like any other column. The demo below shows an example of this.

One few things to note about the demo: The <th> is defined as:

<th rx-status-header>
    <rx-sortable-column
        sort-method="sortcol(property)"
        sort-property="status"
        predicate="sort.predicate"
        reverse="sort.reverse">
    </rx-sortable-column>
</th>

Note that sort-property="status" is referring to the server.status property on each row. Thus the sorting is done in this example by the status text coming from the API.

Usage

as attribute
<ANY rx-status-header>
   ...
</ANY>
