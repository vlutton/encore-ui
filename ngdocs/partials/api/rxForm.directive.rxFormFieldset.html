rxFormFieldset
directive in module rxForm

Deprecated API

Description

DEPRECATED - This directive has been marked as deprecated and will be removed in a future release of the EncoreUI framework. Please see current rxForm documentation for updated functionality.

This directive is used to wrap a set of input fields in the proper HTML

Usage

as element:
<rx-form-fieldset
       legend="{String}"
       description="{String}">
</rx-form-fieldset>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
legendString
  • Text to use for
descriptionString
